Használat

A program használatával kapcsolatos tudnivalók.

1. A program jelenleg háromféleképpen aktiválható:
    - A Tálcán a Képernyõ-billentyûzetre kattintunk.
    - Az Alt + Tab billentyûkombináció lenyomásával.
    - Ha a Beállítások menüben 'A háttér aktiválhatja a programot' menüpont aktív, akkor a háttérre (nem gombra) való kattintással.


2. A menü elrejtéséhez, illetve megjelenítéséhez a CTRL + M billentyû-kombinációt használjuk.

3. A címsor elrejtéséhez, illetve megjelenítéséhez a CTRL + C billentyû-kombinációt használjuk.

4. Az Exit gombot a programból való kilépéshez használhatjuk.

5. A program jelenleg csak az 1280 x 1024-es, és az 1024 x 768-as felbontást támogatja.

6. Amennyiben probléma lépne fel a megjelenítéssel, lépjen ki a programból, törölje a C:\Program Files\PTS Technology\KipBill\KepBill.ini fájlt, majd indítsa újra a programot.

7. A readme.txt fájlban olvashatóak a program jelenlegi hibái, esetleges javaslatok, hogy milyen irányba fejlõdjön a program.

8. A program letölthetõ a http://mortens.uw.hu/Prg/KipBill.zip hivatkozásra kattintva. A program új verziója is ide kerül majd fel.
